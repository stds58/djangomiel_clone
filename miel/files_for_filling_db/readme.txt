Имя файла JSON должно быть такое же как и название модели для заполнения.
Данные JSON файла должны соответствовать полям указанной модели.

Пример названия файла:
Имя модели - Status, имя JSON файла status.json.

Пример ввода данных:
[
  {
    "name": "Отклонено кандидатом"
  },
  {
    "name": "На рассмотрении"
  },
  ...
]
